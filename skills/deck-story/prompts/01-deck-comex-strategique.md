# Prompt 1 — Deck COMEX stratégique (Minto)

**Use case** : défendre une décision stratégique au CODIR/COMEX (budget, pivot, acquisition, recrutement clé).

**Framework** : Pyramide de Minto — answer first → 3 arguments → preuves.

**Audience type** : CEO + directeurs. Durée : 20-30 min.

---

## Le prompt

```
Rôle : tu es un directeur stratégique senior qui prépare un deck pour le COMEX.

Mission : générer un deck de 8 slides pour [ENTREPRISE] afin d'obtenir [DÉCISION ATTENDUE : ex. validation budget IA 2027 de 500K€].

Audience : [CEO / CFO / COO / Directeur métier]. Biais et préoccupations : [ex. CFO sceptique sur le ROI, CEO pro-IA mais veut des preuves chiffrées].

Contexte : [coller ici les données brutes — chiffres, contexte marché, constats internes, enjeux].

Structure Minto :
- Slide 1 : Answer first — la recommandation en 1 phrase
- Slide 2 : les 3 arguments clés (pyramide)
- Slides 3-5 : 1 argument + preuves/data par slide
- Slide 6 : risques identifiés et mitigation
- Slide 7 : plan d'exécution 90 jours
- Slide 8 : ask + next steps + décision demandée

Pour chaque slide, produis :
1. Action title — phrase complète (max 12 mots) qui porte le message
2. 3 bullets max (max 12 mots chacun)
3. Data callout — chiffre choc + source
4. Speaker note 30 secondes

Contraintes :
- Zéro jargon
- 1 slide = 1 message
- Headline test : un CEO qui lit uniquement les titres doit comprendre toute la reco
- Mots interdits : disruptif, révolutionnaire, innovant, synergies, game-changer, next-gen, "fiers de présenter"
```

## Exemple d'input à remplacer

- **Mission** : valider budget IA 2027 de 500K€
- **Audience** : CEO + CFO + COO (PME industrie 200 personnes)
- **Contexte** : 15 cas d'usage IA identifiés, 3 quick wins livrés, concurrent en avance de 18 mois, CFO demande preuve ROI 12 mois

## Exemple d'output attendu (slide 1)

**Action title** : "500K€ en IA en 2027 ramènent 1,8M€ en productivité d'ici 18 mois"
