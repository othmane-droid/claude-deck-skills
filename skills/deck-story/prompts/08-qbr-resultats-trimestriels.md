# Prompt 8 — QBR résultats trimestriels (5S)

**Use case** : revue trimestrielle interne ou client (QBR — Quarterly Business Review).

**Framework** : 5S — Stats, Successes, Struggles, Strategy, Support needed.

**Audience type** : management + parties prenantes. Durée : 60 min.

---

## Le prompt

```
Rôle : tu es un account manager / directeur qui présente la revue trimestrielle.

Mission : générer un deck QBR de 10 slides pour [PÉRIODE — ex. Q1 2026] chez [CLIENT ou DÉPARTEMENT].

Audience : [STAKEHOLDERS]. Objectif : alignement sur les résultats, les difficultés, les priorités du prochain trimestre.

Contexte : [coller KPIs vs objectifs, réalisations, blocages, contexte marché trimestre].

Structure 5S :
1. Cover — période + scope
2. Stats — les 5 KPIs clés vs objectifs (atteints / partiels / ratés) — visuel scorecard
3. Successes — 3 réalisations majeures avec impact chiffré
4. Successes — detail sur la réalisation #1 (preuves, résultats, apprentissages)
5. Struggles — 3 blocages ou ratés, avec root cause analysis
6. Struggles — plan de correction pour le trimestre suivant
7. Strategy — les 3 priorités Q+1 avec objectifs SMART
8. Strategy — roadmap visuelle Q+1
9. Support needed — ce dont tu as besoin de l'audience pour réussir Q+1
10. Next milestones — dates clés + gouvernance

Pour chaque slide :
1. Action title qui porte le verdict (pas "Résultats Q1" mais "On a dépassé 4 objectifs sur 5")
2. Visuel data (scorecard, graphique, heatmap)
3. 3 bullets max
4. Speaker note 45 secondes

Contraintes :
- Transparence totale sur les échecs (ça crédibilise les succès)
- Chiffres vs objectifs, pas juste chiffres absolus
- "Support needed" = demandes concrètes, pas vague
```

## Exemple d'input à remplacer

- **Période** : Q1 2026
- **Département** : Sales
- **KPIs** : ARR 1,2M€ vs 1M€ ✅, cycle 42j vs 35j ❌, NPS 8,1 ✅
- **Successes** : 3 ETI tier 1 signés, ABM lancé
- **Struggles** : cycle allongé, concurrent agressif

## Exemple d'output attendu (slide 1)

**Action title** : "QBR Sales Q1 2026 — 4 objectifs sur 5 atteints, 1 défi à adresser"
