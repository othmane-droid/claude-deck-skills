# Prompt 5 — Deck sales B2B (Before-After-Bridge)

**Use case** : rendez-vous commercial, obtenir un POC ou une signature.

**Framework** : Before-After-Bridge — douleur actuelle → résultat désiré → ton offre = pont.

**Audience type** : buyer économique + buyer technique. Durée : 30-45 min.

---

## Le prompt

```
Rôle : tu es un commercial senior qui présente une offre en rendez-vous découverte.

Mission : générer un deck de 8 slides pour vendre [OFFRE] à [PROSPECT] du secteur [SECTEUR].

Audience : [BUYER ÉCO + BUYER TECHNIQUE]. Objectif : obtenir un POC / signature / second rendez-vous.

Contexte : [douleurs identifiées lors du call découverte, enjeux business prospect, compétition connue].

Structure Before-After-Bridge :
1. Cover — ta boîte × prospect (logos)
2. BEFORE — la réalité douloureuse du prospect (verbatim call découverte + chiffres)
3. BEFORE — le coût de l'inaction chiffré
4. AFTER — la nouvelle réalité possible (vision)
5. AFTER — preuves par les cas clients similaires (résultats quantifiés)
6. BRIDGE — ton offre et comment elle amène de BEFORE à AFTER
7. BRIDGE — délai, garanties, conditions
8. Next step — proposition d'engagement concrète (POC, contrat, pilote)

Pour chaque slide :
1. Action title orienté bénéfice prospect
2. Max 3 bullets visuels (icônes autorisées)
3. Une preuve chiffrée par slide
4. Speaker note 60 secondes

Contraintes :
- Slide 2 doit reformuler les mots exacts du prospect (verbatim)
- Slides 4-5 doivent être chiffrées (pas de "amélioration significative")
- Slide 7 doit inclure une garantie concrète
- Slide 8 = une proposition actionnable, pas "on reste en contact"
```

## Exemple d'input à remplacer

- **Offre** : AI PARTNER™ (audit + build + scale IA)
- **Prospect** : ETI industrie auto, 400 salariés, 80M€ CA
- **Douleurs call** : "on décroche sur l'IA", "concurrence chinoise arrive", "équipes bloquées sur manuel"
- **Compétition** : Accenture (trop cher), freelances (trop dispersés)

## Exemple d'output attendu (slide 3)

**Action title** : "L'inaction vous coûte 2,4M€ par an en productivité manquée"
