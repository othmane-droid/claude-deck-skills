## Comment les utiliser

1. Choisis le prompt qui correspond à ton besoin
2. Avant le prompt, colle ton **bloc charte graphique** (voir `charte-graphique.example.json`)
3. Remplace les `[CROCHETS]` par tes vraies données
4. Colle le tout dans Claude (claude.ai ou Claude Code)
5. Itère 2-3 fois minimum avant de valider
